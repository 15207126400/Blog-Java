# Blog-Java
博客后端管理系统

# 项目介绍
- 核心框架: Spring Boot 2.2
- 持久层框架: Mybatis Plus 3.0
- 数据库: MySQL5.7 Redis
- 数据库连接池: Alibaba Druid
- 代码简化: Lombok
- 文件上传: Jsch
- 富文本插件: Simditor
- 安全框架: Shiro
- 日志管理: Slf4J
- 模板引擎: Thymeleaf
- 存储系统: Minio
